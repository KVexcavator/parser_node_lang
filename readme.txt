Протестировано на Ubuntu 18.04. Node v12.19.0.
В терминале:
- Перейти в папку, cd parser;
- Инициализировать npm, npm i;
- Запустить парсер, node cambridge_parser_sound.js;
- Ответить на всплывающие вопросы, слово вводить без пробелов, 
в нижнем регистре, латинскими буквами. 
- Получить ссылку для скачивания аудиофайла, или скачать во всплывшем окне.

Дополнительно видео how_parser
